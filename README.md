## 📖 Обзор проекта

**Анализ продаж интернет-магазина** — это полноценное веб-приложение для бизнес-аналитики, разработанное на фреймворке Streamlit. Оно позволяет пользователям:

- 📤 **Загружать данные** из CSV и Excel файлов
- 🔍 **Фильтровать** по множеству параметров
- 📊 **Визуализировать** данные с помощью интерактивных графиков
- 📥 **Экспортировать** результаты в CSV и Excel
- ➕ **Добавлять** новые записи через удобную форму
- 📈 **Анализировать** тренды и закономерности

Приложение разработано в рамках экзаменационного проекта и демонстрирует навыки работы с Python, обработкой данных и созданием веб-интерфейсов.

## 🚀 Ключевые возможности

### Быстрый старт
- ⚡ **Загрузка за 3-4 секунды** — оптимизированная обработка данных
- 🎨 **Красивый интерфейс** — современный дизайн с градиентами и анимациями
- 📱 **Адаптивный дизайн** — корректно отображается на всех устройствах

### Работа с данными
- 📤 **Загрузка CSV/Excel** — поддержка файлов любого размера
- 🔍 **Многоуровневая фильтрация** — по категории, городу, дате, поиск по товару
- ➕ **Добавление записей** — через форму с валидацией
- 📊 **Автоматический расчет** — выручки, среднего чека, топ-менеджеров

### Визуализация
- 📊 **4 типа графиков** — столбчатые, круговые, линейные
- 🎯 **Интерактивность** — наведение, увеличение, выделение
- 📋 **Детальная таблица** — все данные в удобном формате

### Экспорт
- 📥 **Скачать CSV** — универсальный формат
- 📊 **Скачать Excel** — профессиональный формат с форматированием
